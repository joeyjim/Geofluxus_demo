# Geofluxus_demo

This project is a React web app scaffold for the geoFluxus map design.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the URL shown in the terminal to view the app.

## Notes

- The initial design uses `src/assets/Map empty.png` as the map background.
- Replace `Map empty.png` with the actual map asset once available.
